Features

Everything you need to manage access

Built for gyms, coworking spaces, and any facility that needs reliable, automated entry control.

👥

User Management

Register members with their name, phone number, and RFID/QR card. Edit profiles, view stats, and manage everything from a clean admin panel.

📦

Flexible Subscriptions

Create custom subscription packages — unlimited, 3x/week, 2x/week, or anything in between. Set duration, weekly session caps, and pricing.

📡

RFID & QR Scanning

Works with physical RFID readers (acts as a keyboard) and the built-in camera QR code scanner. Real-time access decisions in milliseconds.

📊

Access Logs & Audit Trail

Every scan is logged with a timestamp, result, and reason. View recent attempts per-user or across your whole facility from the admin dashboard.

🗓️

Class Scheduling

Define class schedules with capacity limits. Enroll members into classes and track participation alongside their general access logs.

🌍

Multi-Language Support

Built with Flask-Babel for full internationalization. Swap languages on the fly without restarting — ideal for bilingual or international venues.

How it Works

Simple. Fast. Reliable.

The whole flow takes less than a second from card tap to access decision.

1

Card is Scanned

A member taps their RFID card on the reader or scans their QR code. The unique ID is sent to the server instantly.

2

Subscription is Checked

The system looks up the user, verifies their active subscription, checks weekly session counts, and evaluates expiry dates.

3

Decision is Displayed

The screen turns green (granted), red (denied), or amber (warning) with the member's name, plan, and relevant context.

4

Attempt is Logged

Every scan — whether granted or denied — is permanently recorded with a timestamp for audit and reporting purposes.

Quick Setup

Running in under 2 minutes

No Docker, no cloud accounts, no configuration headaches. Clone, install, run.

  • ✓ The SQLite database is auto-created on first launch — no setup needed
  • ✓ Ships as a cross-platform desktop app via PyWebView + PyInstaller
  • ✓ Works on your local network — accessible from any device in the same venue
  • ✓ Works with any standard USB RFID reader (keyboard-emulation type)
Terminal bash 
# 1. Clone the repo
git clone https://github.com/prundusdanielioan/accessControl.git
cd accessControl

# 2. Create & activate venv
python3 -m venv venv
source venv/bin/activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Launch 🚀
python3 app.py
